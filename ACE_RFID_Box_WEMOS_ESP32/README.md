# ACE RFID Box - ESP32 LCD

Author: Bin Sun

This folder contains the original WEMOS/Lolin ESP32 build using a 16x2 I2C LCD, PN532 reader, rotary encoder, read/write buttons, buzzer, and optional external LED.

For wiring, controls, workflows, and troubleshooting, see [USER_MANUAL.md](USER_MANUAL.md).
