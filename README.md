# ACE RFID Box

Author: Bin Sun

Arduino RFID filament tag manager for the ESP32 LCD build.

## Version

- `ACE_RFID_Box_WEMOS_ESP32`: ESP32 with 16x2 I2C LCD. See the [LCD user manual](ACE_RFID_Box_WEMOS_ESP32/USER_MANUAL.md).
