# ACE RFID Box

Author: Bin Sun

Arduino RFID filament tag manager for ESP32-based builds.

## Versions

- `ACE_RFID_Box_WEMOS_ESP32`: ESP32 with 16x2 I2C LCD
- `ACE_RFID_Box_WEMOS_ESP32_SH1106`: ESP32 with SH1106 OLED
