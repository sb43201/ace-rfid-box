# ACE RFID Box

Author: Bin Sun

ACE RFID Box is an ESP32-based Anycubic ACE filament RFID tag manager.

## Versions

- `ACE_RFID_Box_WEMOS_ESP32_SH1106` - WEMOS/Lolin ESP32 version using an I2C SH1106 OLED display.
- `ACE_RFID_Box_WEMOS_ESP32` - ESP32 16x2 LCD version, kept in this branch for reference.
