# ACE RFID Box

Author: Bin Sun

Arduino RFID filament tag manager for ESP32-based builds.

## Versions

- `ACE_RFID_Box_WEMOS_ESP32`: ESP32 with 16x2 I2C LCD. See the [LCD user manual](ACE_RFID_Box_WEMOS_ESP32/USER_MANUAL.md).
- `ACE_RFID_Box_WEMOS_ESP32_SH1106`: ESP32 with SH1106 OLED. See the [SH1106 user manual](ACE_RFID_Box_WEMOS_ESP32_SH1106/USER_MANUAL.md).
