# 3D Printed Case

Place STL files for the ACE RFID Box enclosure in the folder matching the branch/build.

## Folders

- `esp32-lcd` - main ESP32 16x2 LCD version
- `esp32-sh1106-oled` - SH1106 OLED version
- `esp8266` - ESP8266 version

Suggested files for each folder:

- `top.stl`
- `bottom.stl`
- `buttons.stl`
- `encoder-knob.stl`
