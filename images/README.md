# Images

Place finished product photos, wiring photos, and build images in the folder matching the branch/build.

## Folders

- `esp32-lcd` - main ESP32 16x2 LCD version
- `esp32-sh1106-oled` - SH1106 OLED version
- `esp8266` - ESP8266 version

Suggested files for each folder:

- `finished-product-front.jpg`
- `finished-product-inside.jpg`
- `wiring-overview.jpg`
